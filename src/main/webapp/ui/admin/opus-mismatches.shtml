Compare the appointment details between the Primary and Additional appointments in Opus. Highlighted cells show mismatched fields. Where there are mismatches, update the appropriate Opus appointment. If you see a Case icon next to the row, take a look at your Active Cases to see if an unclosed case is the cause of the disparity. The comparison report only includes faculty with joint or split appointments. The report does not include any pending (prospective) appointments. If you have a faculty member with a joint or split appointment and you don't see them listed here, contact opushelp@ucla.edu.
